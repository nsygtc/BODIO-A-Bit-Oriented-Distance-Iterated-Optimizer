## Overview

BODIO is a bit-oriented distance-iterated optimizer for key search under bitwise linear leakage. It reformulates key search as a high-dimensional binary optimization problem and introduces bit-level search, hierarchical guidance, individual diversification, and one-bit-flip refinement.
